# Various_Linear_Regression_Factors
Forecasting apartment prices by analyzing the impact of various factors using linear regression.

# Фархат Фахрутдинов, f.fakhrutdinov@innopolis.university  
Данная часть описывает мою зону ответственности в рамках командного проекта. Моей основной задачей был сбор данных о квартирах, их первичная предобработка, очистка и разведочный анализ данных (EDA).

## 1. Сбор и парсинг данных
* Написан парсер для обработки сырых данных из JSON-файлов, распределенных по различным директориям.
* Извлечены базовые характеристики недвижимости: количество комнат (включая студии), площадь, этаж, общее количество этажей в доме и стоимость.
* Дополнительно собраны признаки на основе текстовых бейджей объявлений: наличие балкона или лоджии, чистовая отделка, статус проверки в Росреестре и признак продажи напрямую от застройщика.
* На основе географических координат (широты и долготы) сформированы пространственные признаки.
* Рассчитано точное расстояние до центра Казани.
* Вычислено минимальное расстояние до ближайшей станции метро с использованием математической формулы для расчета дистанции по сфере.
* Результаты парсинга собраны и сохранены в структурированный файл `kvartiras.csv`.

## 2. Предобработка данных
* Проведена проверка качества данных из файла `kvartiras.csv`.
* Выявлено и удалено более 11 тысяч полных дубликатов для предотвращения утечек и искажений.
* Сгенерирован новый признак «Относительный этаж», отражающий позицию квартиры по отношению к общему количеству этажей в здании.
* Создана целевая переменная `Цена_log` с помощью логарифмирования стоимости. Это преобразование было выполнено ради сохранения редких видов квартир. 
* Для получения финальной цены в рублях из предсказаний модели необходимо использовать обратное экспоненциальное преобразование `np.expm1(предсказание)`.

## 3. Разведочный анализ данных (EDA)
* Признаки были автоматически типизированы и разделены на 11 числовых и 2 категориальных.
* Изучены уникальные значения и распределения категориальных признаков.
* Выполнен анализ выбросов для ключевых числовых признаков: площади, расстояния до центра, расстояния до метро и цены.
* Детектирование аномалий проводилось методом межквартильного размаха (IQR) с консервативным коэффициентом 4.
* Построены boxplot-графики для визуализации распределений и наглядно определены верхние границы нормы, по которым будет проводиться отсечение аномалий.

## Структура файлов моей части
* `kvartiras.csv` — датасет с исходными собранными данными о квартирах.
* `dataset_sample_{1000/3000/5000}.csv` — файлы с уменьшенным количеством данных на проверке моделей линейной регрессии.
* `dataset_sample_all.csv` — общий файл датасета для последующего моделирования.
* `parse_data.py` – скрипт для парсинга json файлов из сайта Avito.
* `create_database.py` – скрипт для извлечения базовых характеристик недвижимости из собранных json файлов.
* `calculate_distances.py` – вспомогательный скрипт, в котором находятся данные о местоположении центра города Казани, местоположении станций метро, а также функции вычисления расстояний между квартирой и целевыми точками.
* `data_process.ipynp' - ipynb файл, в котором визуализированы распределения данных, зависимости между ними, исследованы преобразования признаков для выявления нелинейных зависимостей.
