# Беляев Андрей, an.beliaev@innopolis.university
Данная часть описывает мою зону ответственности в рамках командного проекта. Моей основной задачей являлась реализация и исследование методов оптимизации для обучения модели линейной регрессии, предназначенной для прогнозирования стоимости квартир.

## 1. Реализация метода стохастического градиентного спуска (SGD)
Реализован алгоритм стохастического градиентного спуска (SGD) для обучения модели линейной регрессии.
Использована функция градиента, разработанная в рамках проекта, на основе которой реализовано итеративное обновление весовых коэффициентов модели.
В процессе реализации:
* выполнена настройка коэффициента скорости обучения (learning rate); 
* построены графики изменения среднеквадратичной ошибки (MSE) в процессе обучения. 
* проведено тестирование алгоритма на подготовленном датасете и выполнена оценка качества модели по метрикам MSE, RMSE и коэффициенту детерминации R².
  
## 2. Реализация регуляризации для SGD
Для предотвращения переобучения реализованы два наиболее распространённых метода регуляризации.
* Добавлена L2-регуляризация, ограничивающая величину весовых коэффициентов модели за счёт штрафа на квадрат их нормы.
* Также реализована L1-регуляризация, способствующая уменьшению влияния несущественных признаков путём зануления части коэффициентов.
* Для каждого метода проведён подбор коэффициента регуляризации λ и выполнено сравнение полученных результатов с классическим SGD без регуляризации.
  
## 3. Реализация адаптивного метода Adam
Реализован алгоритм адаптивной оптимизации Adam.
В отличие от классического градиентного спуска, данный метод автоматически изменяет величину шага обучения для каждого параметра модели на основе оценок первого и второго моментов градиента.
В процессе экспериментов:
* реализовано обучение модели методом Adam; 
* исследована скорость сходимости алгоритма; 
* построены графики изменения функции ошибки по эпохам; 
* выполнено сравнение качества обучения с методом SGD.
  
## 4. Реализация регуляризации для Adam
Для алгоритма Adam реализованы обе разновидности регуляризации:
* L2-регуляризация; 
* L1-регуляризация. 
Для каждого варианта произведён подбор значения коэффициента λ и исследовано влияние регуляризации на итоговое качество модели.
Полученные результаты были сопоставлены с классическим Adam без использования регуляризации.

## 5. Сравнение методов оптимизации
Проведено комплексное сравнение реализованных алгоритмов:
* SGD; 
* SGD + L2; 
* SGD + L1; 
* Adam; 
* Adam + L2; 
* Adam + L1.
* 
Для каждого метода вычислены основные показатели качества модели:
* среднеквадратичная ошибка (MSE); 
* корень из среднеквадратичной ошибки (RMSE); 
* коэффициент детерминации (R²).
  
Результаты представлены в виде сравнительной таблицы и графиков, позволяющих оценить влияние используемого метода оптимизации и регуляризации на качество прогнозирования стоимости квартир.
